Project Proposal

This project is a cloud-hosted web application titled “Local Bird Sightings”, deployed on a Microsoft Azure virtual machine using an Infrastructure as a Service (IaaS) model.

The purpose of the system is to provide a simple, accessible platform for recording and sharing local bird observations, including species identification, location, date, and descriptive notes.

The server environment was deployed using Microsoft Azure, Ubuntu Linux, Apache2 web server software, and Bash scripting technologies.

The website demonstrates practical cloud infrastructure deployment, Linux administration, website hosting, and scripting concepts developed throughout ICT171.
