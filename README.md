## Project Overview

This project demonstrates deployment of a cloud-hosted website using Microsoft Azure Infrastructure-as-a-Service (IaaS), Ubuntu Linux, Apache2 web server technology, Bash scripting, and HTML/CSS web development.

The website provides a lightweight platform for displaying local bird sightings, species information, and server-hosted sighting logs.

- Ubuntu Linux
- Apache2 Web Server
- Bash Scripting
- HTML5
- CSS
- GitHub Version Control

---

## Infrastructure Overview

The project environment was deployed using a Microsoft Azure virtual machine running Ubuntu Linux.

Key infrastructure components include:
- Azure Virtual Machine
- Public IP Address
- Apache2 Web Server
- SSH Remote Administration
- Azure Networking Security Rules
- HTTPS Preparation using Certbot

---

## Website Features

- Bird sighting records
- Bird species listings
- Sighting statistics
- Weather information links
- Live sightings log
- Creative Commons licensing information

---

## Bash Automation Script

A Bash script named `add_sighting.sh` was developed to automate the recording of bird sightings within the Linux server environment.

The script:
- accepts user input
- records bird sighting details
- appends information to a server-hosted sightings log
